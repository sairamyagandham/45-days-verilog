# Day 06 - 4:2 Encoder

## Description
Implemented a 4:2 Encoder using Verilog HDL.

## Concept
A 4:2 Encoder converts 4 input lines into a 2-bit binary output.

### Truth Table

| Input | Output |
|--------|--------|
| 0001 | 00 |
| 0010 | 01 |
| 0100 | 10 |
| 1000 | 11 |

> Note: Only one input should be HIGH at a time.
