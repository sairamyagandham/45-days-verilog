# Day 16 - 4-to-2 Encoder

## Description
Implemented a 4-to-2 Encoder using Verilog HDL.

## Concept
An encoder converts multiple input lines into a smaller number of output bits.

This encoder converts 4 input lines into 2-bit binary output.

## Inputs
- in[3:0]

## Outputs
- y[1:0]
