# Day 28 - Mealy FSM Basics

## Description
Implemented a basic Mealy Finite State Machine (FSM) using Verilog HDL.

## Concept
A Mealy FSM is a sequential circuit where the output depends on:
- Present State
- Current Input

## Inputs
- clk
- in

## Output
- out

## FSM Features
- State Transition
- Input-dependent Output
- Sequential Logic
