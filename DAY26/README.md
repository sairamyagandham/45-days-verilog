# Day 26 - 4-bit Down Counter

## Description
Implemented a 4-bit Down Counter using Verilog HDL.

## Concept
A Down Counter decrements its value by 1 on every positive edge of the clock.

The counter counts in reverse binary order.

## Input
- clk

## Output
- q[3:0]

## Counting Sequence
1111 → 1110 → 1101 → 1100 → 1011 → ...
