# Day 27 - Mod-10 Counter

## Description
Implemented a 4-bit Mod-10 (Decade) Counter using Verilog HDL.

## Concept
A Mod-10 Counter counts from 0 to 9 and then resets back to 0.

It is also known as a Decade Counter.

## Input
- clk

## Output
- q[3:0]

## Counting Sequence
0000 → 0001 → 0010 → ... → 1001 → 0000
