# Day 20 - SISO Shift Register

## Description
Implemented a 4-bit SISO (Serial In Serial Out) Shift Register using Verilog HDL.

## Concept
A shift register shifts data serially from one flip-flop stage to another on every clock pulse.

SISO means:
- Serial Input
- Serial Output

## Inputs
- clk
- data_in

## Output
- q[3:0]
