# Day 19 - T Flip-Flop

## Description
Implemented a T Flip-Flop using Verilog HDL.

## Concept
A T Flip-Flop toggles its output when T = 1 during the positive edge of the clock.

When T = 0, the output remains unchanged.

## Inputs
- clk
- t

## Output
- q

## Truth Table

| T | Operation |
|---|---|
| 0 | Hold |
| 1 | Toggle |
