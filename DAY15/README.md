# Day 15 - Simple ALU

## Description
Implemented a Simple ALU (Arithmetic Logic Unit) using Verilog HDL.

## Operations Performed

| Select | Operation |
|---|---|
| 00 | Addition |
| 01 | Subtraction |
| 10 | AND |
| 11 | OR |

## Inputs
- a[3:0]
- b[3:0]
- sel[1:0]

## Output
- result[3:0]
